Feedback Machine 2.5

Add headings to categorize your feedback, then select a heading when adding a comment. Check the headings and comments you wish to paste into the student feedback, then click COPY TEXT. You can reset the checkboxes between students by clicking RESET SELECTIONS, or simply uncheck the ones that do not apply.

To add a new folder/file: name the folder with only 2 characters, then prefix each file name in that folder with those 2 characters... WD/WD-Assign-1.js
